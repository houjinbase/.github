# Houjinbase

Houjinbase は、日本法人に関する公開情報を検索・確認できる法人情報データベースです。  
法人番号、所在地、商号変更、移転、閉鎖、公開調達情報、特許・届出・認定など、公開データをもとに企業調査や市場分析に役立つ情報を整理しています。

Website: https://www.houjinbase.com/

---

## Houjinbase について

Houjinbase は、日本国内の法人情報をより探しやすく、比較しやすくすることを目的とした公開データ活用プロジェクトです。

主な用途：

- 日本法人の基本情報確認
- 企業の所在地・商号・法人番号の確認
- 新設法人、移転、商号変更、閉鎖などの動向確認
- 公共調達、特許、届出・認定などの公開情報の調査
- B2B 営業、取引先確認、市場調査、リスク確認の補助

---

## 主な機能

- 法人名・法人番号による法人検索
- 都道府県・市区町村別の法人情報閲覧
- 法人詳細ページ
- 月次の法人動向レポート
- 公共調達データレポート
- gBizINFO 関連データのレポート
  - 特許
  - 届出・認定
  - 公共調達
- データレポートによる地域別・年度別の傾向確認

---

## データソース

Houjinbase では、主に以下の公開データをもとに情報を整理しています。

- 国税庁法人番号公表サイト
- gBizINFO
- その他、政府・公的機関が公開する法人関連データ

各データは公開情報に基づいており、Houjinbase はそれらを検索・閲覧・分析しやすい形に整理することを目指しています。

---

## 更新頻度

データは原則として **毎日更新** されます。

ただし、各データソース側の公開タイミング、データ形式、取得状況により、実際の反映タイミングには差が生じる場合があります。
